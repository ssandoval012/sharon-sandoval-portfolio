ASME IMECE2025-166047  ·  First Author

Smart Manufacturing Using Advanced Technologies for Human Robot Collaboration

Sandoval, S., Guerra-Zubiaga, D.A., Richards, G., Kuts, V. — ASME International Mechanical Engineering Congress & Exposition, 2025

Proposed a machine-learning-informed digital twin model — built in Siemens NX and Siemens Tecnomatix Process Simulate — for the smart manufacturing of Phase Change Materials (PCMs). The model integrates collaborative robot workflows, an IIoT sensor framework, and a proposed CNN trained on historical process data to predict potential failures during simulation and manufacturing. A pragmatic, real-world implementation path was also specified, detailing a Kawasaki CL110N cobot fitted with heat-resistant grippers for handling molten salt hydrate up to 100°C under ISO/TS 15066-compliant safety behavior.

Improving Efficiency of Deep-Learning based Vision Transformer for Remote Sensing for Satellite Images

Adeogun, D., Sandoval, S., Wulfekuhle, V., Vu, K., Kouwenhoven, F. — Kennesaw State University, 2026

Optimized a Vision Transformer for satellite image classification on EuroSAT — 27,000 real, georeferenced Sentinel-2 satellite images across 10 land-use classes. Applied transformer layer reduction, weight pruning, response-based model distillation, and post-training / quantization-aware compression (PTQ, QAT, ONNX static) to reduce computational cost while preserving classification accuracy.

Control Methods for Driving a Robotic Manipulator with a Nonlinear Model

Arellano, J., Sandoval, S., Smith, D. — Kennesaw State University

Developed a nonlinear controller for a two-joint prismatic manipulator, comparing PD control, inverse dynamics control with feedback linearization, and quadratic-programming-based torque optimization. Demonstrated that PD control alone fails under gravity loading, while feedback linearization achieves zero steady-state error — and that constrained optimization keeps joint torques within actuator limits without sacrificing controller performance.

An Exploration of Control Methods for a Robotic Joint with Harmonic Gears

Arellano, J., Sandoval, S., Smith, D. — Kennesaw State University

Compared PD control, feedforward control, and state-space control (with integral action and observer design) for a harmonic-geared robotic joint. Derived the joint's transfer function and full state-space model from first principles, then evaluated each controller's step response for settling time, overshoot, and steady-state error across constant and non-constant reference trajectories.

Forward and Velocity Kinematics of a 4-Joint Robotic Arm

Arellano, J., Sandoval, S., Smith, D. — Kennesaw State University

Manually derived the forward kinematics of a 4-joint robotic arm using the Denavit–Hartenberg convention, then formulated the Jacobian matrix to analyze end-effector velocities and identify five distinct singularity configurations. Validated all manual calculations against simulation in Peter Corke's MATLAB Robotics Toolbox across five joint configurations, with results matching to four significant figures.

An Experiment with a Partitioned Visual Servoing Controller

Arellano, J., Sandoval, S., Smith, D. — Kennesaw State University

Designed and validated a partitioned Image-Based Visual Servoing (IBVS) controller for a robotic arm's end-effector-mounted camera. Performed physical camera calibration using a real webcam and checkerboard pattern, then demonstrated that decoupling z-axis rotation from translational motion allows the camera to converge on a target pose under 180° rotation — a case where traditional IBVS fails outright.

Coursework — CS 7267 Machine Learning

Unsupervised & Supervised Learning — Clustering and Classification

Sandoval, S. — Kennesaw State University

Applied principal component analysis to reduce a 64-dimensional handwritten digit dataset to 2 components, then clustered the reduced data with K-Means and Gaussian Mixture Models, evaluating cluster quality against true digit labels via silhouette score. Optimized the number of PCA components (tested 2, 4, 6, 8, and 10) and validated clustering behavior on a synthetic blob dataset. Separately applied supervised classifiers — Naive Bayes and K-Nearest Neighbors — to a held-out test split, tuning K and handling continuous variables to improve performance, then extended both classifiers to the Wisconsin breast cancer dataset.
